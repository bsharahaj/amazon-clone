# Amazon Product Page Clone

A pixel-perfect clone of an Amazon product page, built with pure HTML and CSS only — no JavaScript, no frameworks.

## 📋 Project Description

This project is a recreation of an Amazon product page for a HUANUO Dual Monitor Stand. It was built as a fullstack exercise to practice HTML and CSS layout techniques, including CSS Grid, Flexbox, and modern positioning (sticky elements).

**Original product page:** [HUANUO Dual Monitor Stand on Amazon](https://www.amazon.com/dp/B07T5SY43L)

## 🎯 Features

- Fully responsive layout
- Sticky product image that follows on scroll
- Sticky buy box (right sidebar) that follows on scroll
- Pixel-perfect Amazon styling (colors, fonts, spacing)
- Image gallery with thumbnails
- Color and size variant selectors
- Specs table and "About this item" section
- Two product carousels ("Deals on related products" and "Products related to this item")
- Custom SVG icons (location pin, search, US flag, shopping cart)

## 🛠️ Built With

- **HTML5** — semantic structure
- **CSS3** — Grid, Flexbox, sticky positioning, custom properties
- **Inline SVG** — for icons
